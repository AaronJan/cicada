# Rock Your Head （码农颈椎操）


码农累了烦了无聊了怎么办？打开PC摄像头，一起Rock your head!

游戏很简单，跟着音乐节奏，当音符到了码农头上就把头摇起来，简直停不下来。

[在线DEMO](http://aaronjan.github.io/rock-your-head/)
